# 🏋️ Sistema de Gestão de Academia

## 📌 Descrição
Sistema desenvolvido em Java utilizando o padrão MVC para gerenciamento de academia, com persistência em banco de dados. Projeto desenvolvido em aula na disciplina de Pesquisa Orientada a Objetos.

---

## 🚀 Tecnologias
- Java
- JDBC
- MySQL
- Swing (JOptionPane)

---

## 🏗️ Arquitetura
- MVC (Model-View-Controller)
- DAO (acesso ao banco de dados)
- ConnectionFactory

---

## ⚙️ Funcionalidades
- Cadastro de usuários
- Cadastro de treinos e exercícios
- Avaliação física (IMC)
- Controle de pagamentos

---

## 🗄️ Banco de Dados
É necessário configurar na classe `ConnectionFactory`:
- URL
- Usuário
- Senha

---

## ▶️ Como executar
1. Criar banco de dados
2. Configurar conexão
3. Rodar o projeto
